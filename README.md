# 🧋 手搖飲大王 (Bubble Tea Tycoon)

**手搖飲大王** 是一款運行於瀏覽器的像素風格經營模擬遊戲（PWA）。玩家將在台北永吉路30巷建立自己的手搖飲帝國，從小型攤販成長為傳奇品牌。

## 🎮 遊戲特色

*   **像素藝術風格**：復古可愛的 8-bit/16-bit 視覺設計。
*   **經營模擬**：管理菜單、調整配方、應對不同類型的顧客（學生、上班族等）。
*   **真實時間機制**：遊戲內時間與現實時間掛鉤（可調整比例），體驗日夜循環與營業時間壓力。
*   **競爭對手**：與 AI 控制的競爭對手（如強尼紅茶公社）爭奪市場份額。
*   **PWA 支援**：支援 Progressive Web App，可安裝於手機桌面，隨時隨地遊玩。

## 🚀 快速開始

### 線上遊玩
(在此處填入您的 GitHub Pages 連結，例如: https://cokefever.github.io/BubbleTeaTycoon/)

### 本地開發

1.  Clone 此專案：
    ```bash
    git clone https://github.com/CokeFever/BubbleTeaTycoon.git
    ```
2.  進入目錄並開啟 `index.html` 即可遊玩（建議使用 VS Code 的 Live Server 擴充功能以獲得最佳體驗）。

## 🛠️ 技術棧

*   **Core**: HTML5, CSS3, JavaScript (ES6+)
*   **Framework**: React 18 (via CDN for simplicity)
*   **Styling**: Tailwind CSS (via CDN)
*   **Icons**: Custom Pixel Art
